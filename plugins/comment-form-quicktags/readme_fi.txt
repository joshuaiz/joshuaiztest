This plugin inserts a quicktag toolbar on the comment form.

== Kuvaus ==

Tämä lisäosa luo tagit kommentointiin. Asetuksia voi säätää ylläpidon asetusvalikon kautta.

= Ominaisuudet =

*   Pystyt muokaamaan asetuksia WordPressin ylläpidon puolelta.

== Asennus ==

1. Lataa ja pura lisäosan tiedostot /wp-content/plugins/-hakemistoon.
2. Aktivoi lisäosa 'Asennetut'-sivulta. Se löytyy valikosta "Lisäosat".
3. Mene "Asetukset"-valikkoon -> Comment Form Quicktags

== Kuvankaappaukset ==

1. Kommentoinnin yläpuolella
2. Comment Form Quicktags asetukset-sivulta
